What is a Data Center and how do they relate to Cloud Technology:

Data centers a massive building that house hundreds of servers for a company or organization. There can be hundreds of servers or one single giant server. These are data housing facilities that store thousands of terabytes of information for companies. There is climate controlled environment, back up power generators anything to keep the servers running.

The Cloud is basically a remote data center that can be in any place in the world and can be accessed through the internet. The cloud can be accessed from anywhere and by anyone who is authorized. This makes group work very easy and accessible. The best example for this would be Google Drive a cloud service that google offers to the public. One of the services included in google drive is google docs, which is basically the online version of Microsoft Word. The catch is that the Google Doc file is not stored on your computer it is stored on google's cloud, so all changes are saved onto the Cloud and don't need you to press save or save as. Also anyone that you give access to can edit the document and it will be saved on googles cloud making joint projects very easy.

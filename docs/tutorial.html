Docker Tutorial:

Downloading Docker:

  1. Download and install Docker on your computer.
    Download Docker
  2. Run the Docker application. When prompted, enable the Enable Hyper-V and the WSL 2 Features.
  3. Finish the installation wizard instructions and close the wizard.
  4. Register a Docker ID on Docker Hub.
    Register ID
  5. Go to the Start menu and start the Docker Application.

Setup and Updates:

  1. Open a "Terminal" by going to the Start Menu and typing "cmd". Click Enter to run the Command Prompt App.
    • Test that Docker is working by running a "docker run hello-world" Command.
  2. Pull the Ubuntu image, which contains the set of instructions needed for creating a container, by running the "docker run -it ubuntu:latest /bin/bash" Command.
    • Note the Container ID name which is displayed, for future use.
  3. Download and update the latest packages for the repositories to get information on the newest versions of packages and their dependencies by running the "apt-get update" Command.
  4. Download and install a text editor to be able to create and edit an HTML file by running the "apt-get install vim" Command.

Create a new Docker Container:

  1. Open a "Terminal" by going to the Start Menu and typing "cmd". Click Enter to run the Command Prompt App.
  2. Check the available Containers by running the "docker ps -a" Command.
  3. Create a new Image by running the "docker commit Container ID Image Name ".
    Replace Container ID with an available Container ID.
    Replace Image Name with the name of your new Image.
  4. Check that your new Image was properly created by running the "docker images" Command.
    • You should see your new Image listed under Repository.

Host a Webpage:

  1. Open a "Terminal" by going to the Start Menu and typing "cmd". Click Enter to run the Command Prompt App.
  2. Start a Host Shell and create a folder on your local drive to host a site by running the "docker run -it -p 127.0.0.1:80:80 -v Local Folder _dir:/var/www/html Image Name /bin/bash" Command.
    Replace Local Folder with the location where you will be storing the html files.
    Replace Image Name with the name of your Image.
  3. Start your Docker Container by running the "docker start Image Name " Command.
  4. Change the current directory so that you can save an HTML filed into the appropriate folder by running the "cd /var/www/html" Command.
    • Your Command Line should now show "root@ Image Name : /var/www/html"
  5. Create a homepage by making an "index.html" file using the "vi index.html" Command.
  6. Edit the new HTML file by pressing the "i" keyboard key.
    • Once you enter Edit mode, you will see "--INSERT--" at the bottom of the screen.
  7. You can now enter the HTML code for your website.
    • Use this handy reference guide for useful shortcut commands.
      Cheat Sheet
  8. Once you are done entering your HTML code, press the "ESC" key on your keyboard and type ":wq" to save the changes.
  9. Start the Web Service by using the "/etc/init.d/apache2 restart" Command.
    • Check that your page is being properly hosted by typing "127.0.0.1" into the address bar of a web browser.

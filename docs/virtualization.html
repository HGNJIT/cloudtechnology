What is Virtualization and Containerization:

Before virtualization, one server was used to do one singular task. An example of this would be if there were three different servers, one would handle spreadsheets, one would handle emails, and the last one would handle writing documents. With virtualization one server can be split up into three parts each handling the task they are supposed to. Virtualization can also be done on the computer you are on right now. Using the hardware you allow it to from your computer, certain programs can have another computer running inside of your computer.

If Virtualization is making a fake computer in side of another computer, than containerization would be making a fake operating system inside of another operating system.
